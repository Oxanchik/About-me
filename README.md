# О СЕБЕ
_Опытный аналитик с экспертизой в данных продаж, ERP-системах и международной аналитике. Специализация: автоматизация отчетности (SQL/VBA), выявление аномалий в финансовых данных, прогнозирование спроса и ценообразование. Владею английским, французским, испанским, русским. В настоящее время прохожу обучение на курсе Python-разработчик: расширенный курс, закончила первый модуль обучения основам языка программирования._
